# LabInBox-Embedded-AI-Microscop
Embedded Intelligent Microscopy System for Identification and Counting of Microscopic Marine Organisms

**❖ IDEA / SOLUTION :**
• LabInBox AI is a portable, intelligent device that
attaches to any standard microscope.
• It uses on-device fully offline AI for real-time
identification and counting of marine
microorganisms, delivering instant results directly
in the field.
• Provides the identification and counting for 127
different marine microorganisms.
• Having 83% (mAP50-95 score)average accuracy
for identification of microorganisms.
• The model trained on 56,000 realistic, perfectly
labeled training images

**❖ Problem Resolution :**
▪ Our solution directly replaces the slow, expensive
and error-prone manual counting process.
▪ It provides an automated and portable system for
water surveys and research purposes in marine
science.
